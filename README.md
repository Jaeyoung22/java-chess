## ✨ 기능 구현 목록
- [ ] 체스보드를 만든다.
  - [ ] 체스보드는 8 * 8 구성되어있다.
  - [ ] 세로는 숫자로(1~8), 가로는 알파벳(a~h)


- [ ] 게임 시작및 종료 명령어를 입력 받는다.
  - [ ] start 는 시작
  - [ ] end 는 종료


- [ ] 기물(Piece)를 배치한다.
  - [ ] 검은색은 대문자로, 흰색은 소문자로
